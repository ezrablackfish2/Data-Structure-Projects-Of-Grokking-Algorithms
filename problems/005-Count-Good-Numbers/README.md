# Count Good Numbers

## Problem
Given an adjacency list graph and a start node, return the BFS visit order.

Input: dict mapping node to list of neighbors, start node.
Output: list of nodes in BFS order.

Example: {"A":["B","C"],"B":["D"],"C":[],"D":[]} -> ["A","B","C","D"]

## Approach
Use a direct, readable implementation based on the problem definition.
